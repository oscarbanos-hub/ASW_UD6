# Backend - Formación Global Online

API backend del proyecto Formación Global Online, esta desarrollada con Node.js, Express y Mongoose.

## Comandos para el desarrollo

Con npm install, instalamos las dependencias necesarias.
Con run dev, arrancamos el servidor en modo desarrollo.
```
npm install
npm run dev
```

El servidor arranca en el puerto 3000 por defecto. Configurado en: `.env`.

## Documentación de la API

Hemos metido swagger para la documentación de la API, además de que nos facilitará la vidapara probar endpoints, puede darnos facilidades con OpenAPI y la generación de clientes, aunque para esta actividad no la usaremos
```
http://localhost:3000/api-docs
```
